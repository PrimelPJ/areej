# Areej — عَرَج
Islamic personal growth web app for young Muslims.

## Features
- 40 Hadith memorisation with flip cards and quiz mode
- Arabic vocabulary learning
- Sunnah habit tracker with sahabi context
- Prophetic anger management guide
- Daily shukr log tied to Quran verses
- Daily challenges
- Badges and streak tracking
- Dark mode
- Google OAuth + email auth
- Progress saved per user across all devices

## Stack
- Vite (vanilla JS)
- Supabase (auth + database)
- Deployed on Vercel

## Local setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file based on `.env.example`
4. Add your Supabase URL and anon key
5. Run `npm run dev`

## Deploy
Push to GitHub — Vercel auto-deploys.
